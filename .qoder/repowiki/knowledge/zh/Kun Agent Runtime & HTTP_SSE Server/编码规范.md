- 所有核心业务接口均定义在 `src/ports` 目录下，并以 `interface` 或抽象类型形式存在，禁止在领域逻辑中直接引用具体适配器类。
- 配置加载统一使用 `zod` Schema 进行严格校验（如 `KunConfigSchema`），并在 `src/config` 中提供默认值合并逻辑。
- Agent 循环中的状态变更（如工具调用结果、文本增量）通过 `RuntimeEventRecorder` 记录并发布到 `InMemoryEventBus`，实现解耦的事件驱动更新。
- 工具调用执行前必须经过 `ApprovalGate` 或 `LocalToolHost` 的策略检查（如 `policy: 'never'` 或 `untrusted`），确保操作安全性。
