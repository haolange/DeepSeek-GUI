- **分层架构**：采用清晰的端口-适配器（Ports & Adapters）模式。`src/ports` 定义核心接口（如 `ModelClient`, `ToolHost`, `SessionStore`），`src/adapters` 提供具体实现（如 `DeepseekCompatModelClient`, `LocalToolHost`, `FileSessionStore`）。
- **核心循环**：`src/loop/agent-loop.ts` 实现 Agent 主循环，负责上下文压缩、模型请求构建、流式响应处理及工具调用调度。
- **服务层**：`src/services` 封装业务逻辑（如 `ThreadService`, `UsageService`），`src/server` 基于 Node.js 原生 HTTP 模块实现 REST/SSE 服务端，通过 `runtime-factory.ts` 进行依赖注入组装。
- **领域模型**：`src/domain` 和 `src/contracts` 定义不可变的数据结构与事件契约，确保状态流转的一致性。
